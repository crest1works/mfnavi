# M-Factory 売場ナビ

商品マスタ管理サイト（静的HTML）。ビルド不要で、Netlifyからそのまま配信できます。

## ページ構成

- `index.html` — トップページ
- `new-products.html` — 新商品一覧
- `price-changes.html` — 価格改定情報
- `notice-softcarry.html` — お知らせ（ソフトキャリー）
- `notice-seal.html` — お知らせ（シール）
- `robots.txt`

## デプロイ

Netlifyでこのリポジトリを連携すると、`main`ブランチへのpushで自動デプロイされます（ビルドコマンドなし、公開ディレクトリはリポジトリ直下）。
